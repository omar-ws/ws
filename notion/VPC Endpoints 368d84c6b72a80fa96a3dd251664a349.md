# VPC Endpoints

Caminho: EC2 ⇒ ROTAS ⇒ internet ⇒ s3.amazon

O amazon ec2 com sua ENI (infertface de rede) so consegue acessar as conexoes locais, a EC2 ta dentro da VPC sua ENI acessa conteudos da VPC por isso uma ENI no VPC endpoint interface e diferente, um S3 ou um DynamoDB esta fora da VPC.

VPC endpoint veio para solucionar o problema de acessar a internet para acessar o servico da propria AWS

VPC ENDPOINT gataway para s3 e dynamo, gratis, dynamo e s3 tem ip previsiveis/estaveis, usa somente as tabelas

VPC ENDPOINT interface para os demais servicos, cria uma placa ENI direto ao servico e usa DNS
