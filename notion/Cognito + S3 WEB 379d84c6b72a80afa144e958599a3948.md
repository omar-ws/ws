# Cognito + S3 WEB

```jsx
cd ~/environment
aws s3 cp website s3://nome-do-bucket/ --recursive --cache-control "max-age=0"

- aws s3 cp → copia arquivos para o S3
- website → pasta local que você quer enviar
- s3://nome-do-bucket/ → destino no S3
- --recursive → envia todos os arquivos dentro da pasta
- --cache-control "max-age=0" → força o CloudFront a não usar cache velho
```

https://docs.aws.amazon.com/cli/latest/reference/s3/cp.html

```jsx
—recursive //sobe o arquivo sem duplicar
```

```jsx
—cache-control "max-age=0" //o cloudfront guarda cache dos arquivos. Se você subir um config.js novo mas o CloudFront ainda tem o antigo em cache, o usuário continua vendo a versão velha.
```
