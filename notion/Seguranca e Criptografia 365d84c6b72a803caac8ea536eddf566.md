AWS Detective: investiga incidentes em ordem cronologia dos principais servicos de Log

AWS Inspector: ve vulnerabilidades da ec2, patches desatualizados e etc

AWS Security Hub: Painel de todos os alertas de seguranca, exemplo o Inspector, GuardDuty, Macie, CloudTrail, cada um gera alertas separados, porem estando no Security Hub voce consegue ver todos os alertas em um unico lugar

AWS Trust Advisor: mostra as recomendacoes do 6 pilares da AWS, depois que voce ativa o AWS Security Hub no Trust Advisor comeca a mostrar os dados do Security Hub tambem
