# Criptografia de Dados em Repouso

a chave assimetrica e usada uma vez para o servidor descobrir a chave simetrica que o navegador criou, depois que ele sabe ele conversam apenas pela chave simetrica

## chave simetrica:

 uma chave apenas de criptografia, exemplo uma chave descriptografa e criptografa

quando usar:

dados dentro da organizacao 

menos processamento computacional

uma das escolhar para criptografar grande quantidade de dados

## chave assimatrica:

duas chaves de criptografia, uma publica, criptografa e um privada, descritografa, um pouco mais lenta

quando usar:

dados fora da organizacao

regulamentos de governanca
