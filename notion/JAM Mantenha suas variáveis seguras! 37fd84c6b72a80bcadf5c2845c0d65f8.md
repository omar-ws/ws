# JAM Mantenha suas variáveis seguras!

visao geral:

```jsx
Plano de fundo
Você ingressou recentemente na ABC Corp como Engenheiro de Segurança. Seu CISO encarregou de reforçar a segurança dos serviços da AWS. Como parte dessa iniciativa, você é solicitado a criptografar as variáveis de ambiente usando o AWS Key Management Service. Isso deve ser feito para se beneficiar de um controle mais granular sobre o processo de criptografia/descriptografia de dados. Você está pronto para o desafio?
```

task 1:

```jsx
Tarefa 1: Criptografar as variáveis em trânsito e repouso
Pontos possíveis
80
Penalidade por pista
0
Pontos disponíveis
0

Tarefas e pistas
Antecedentes
Como profissional da AWS e em seu primeiro dia de trabalho, seu gerente solicita que você criptografe variáveis de ambiente de uma função lambda em trânsito e em repouso usando uma chave KMS gerenciada personalizada.

Tarefa
Sua tarefa é criptografar as variáveis usando uma chave mestra existente do cliente.

Começando
Abra o AWS Console. Você pode usar esta documentação para se orientar no console: https://docs.aws.amazon.com/lambda/latest/dg/configuration-envvars.html

Inventário
Trabalhe com os recursos Lambda e KMS abaixo.

Lambda - myLambdaFunction.
KMS - myKeyAlias.
Serviços que você deve usar
Lambda, KMS.

Validação de tarefas
A tarefa será concluída automaticamente quando você encontrar a solução. Além disso, você sempre pode verificar seu progresso pressionando o botão “Verificar meu progresso” na tela de detalhes do desafio.
```
