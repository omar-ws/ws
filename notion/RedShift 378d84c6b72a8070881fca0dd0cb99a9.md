maneira do JAM:

```sql
COPY nome_da_tabela
FROM 's3://nome-do-bucket/pasta/'
IAM_ROLE 'arn:aws:iam::123456789:role/NomeDaRole'
DELIMITER '|' <- se o arquivo usar pipe
IGNOREHEADER 1;  <= se tiver cabecalho na primeira linha
```
